Welcome to PegToken
=============================================

Peg is a library for writing secure Smart Contracts on Ethereum.

With Peg, you can build distributed applications, protocols and organizations:

* using :doc:`contract-security-patterns`
* in the `Solidity language <https://solidity.readthedocs.io/en/develop/>`_.
